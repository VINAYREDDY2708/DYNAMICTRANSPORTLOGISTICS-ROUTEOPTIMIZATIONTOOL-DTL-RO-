🚚 Dynamic Transport Logistics & Route Optimization Tool (DTL-RO)
📌 Project Overview

This project is a Dynamic Transport Logistics and Route Optimization Tool designed to improve delivery efficiency by optimizing routes and assigning orders to vehicles intelligently.

It helps reduce:

Travel time
Fuel consumption
Operational cost

The system automates logistics planning and eliminates manual route selection.

🎯 Objective

The main goal of this project is to:

Automate delivery route planning
Assign orders to vehicles based on capacity
Minimize total travel distance
Improve delivery efficiency
⚙️ Features
📦 Add and manage delivery orders
🚛 Assign orders to vehicles
🗺️ Route optimization using nearest distance logic
🔄 Dynamic updates (add new orders anytime)
📊 Displays optimized routes and total distance
💡 Supports single-route and multi-route modes
🧠 Algorithm Used

The project uses a Nearest Neighbor-based heuristic approach:

Select nearest delivery location
Check vehicle capacity
Assign order
Repeat until all deliveries are completed
